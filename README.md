# Welcome to Oribtal Scope
## Tech Stack:
![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=for-the-badge&logo=javascript&logoColor=%23F7DF1E) ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) 

Our team has designed a web application that visualizes Earth-orbiting satellites in real time.
Users can view orbital parameters, satellite metadata, and predicted ground tracks for individual objects.
Data updates continuously, and objects can be filtered by operator, orbit type, or mission category.
